# IPL Data Analysis Dashboard (Tableau)

# Overview
This project presents a comprehensive **IPL (Indian Premier League) Data Analysis Dashboard** built using **Tableau**.  
The dashboard provides interactive insights into **team performance, player statistics, match outcomes, venue analysis, and toss impact**.

The objective of this project is to convert raw IPL match data into **meaningful visual insights** that help understand trends and patterns in cricket analytics.

# Dashboard Features

### 🏆 Tournament Overview
- Displays **Title Winner** for selected season
- Highlights **Orange Cap (Top Run Scorer)**  
- Highlights **Purple Cap (Top Wicket Taker)**
- Total **Tournament 6’s and 4’s**

- Tree map visualization of venues

# Geo-Spatial Analysis
